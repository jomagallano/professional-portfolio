# Jo Magallano — Professional Consulting Platform

> **Closing the gap between AI potential and organizational readiness.**

This repository contains the source code for my professional consulting platform, built to showcase AI enablement strategies, literacy frameworks, and strategic readiness for the ASEAN enterprise and public sectors.

## 🎯 The Vision
Modern organizations often face an "execution gap" where technology outpaces people. This platform highlights a **Human-Centered Intelligence** approach, using systems thinking to bridge that gap and drive grassroots competency.

## 🛠️ Tech Stack
* **Frontend:** Vanilla HTML5, CSS3 (Custom variables, Glassmorphism, CSS Animations)
* **Logic:** JavaScript (Intersection Observer API for scroll animations)
* **Typography:** Cormorant Garamond (Serif) & DM Sans (Sans-serif)
* **Deployment:** [Vercel](https://vercel.com)

## 📁 Key Features
* **Boutique Aesthetic:** A refined plum and gold palette designed for high-end advisory optics.
* **Responsive Architecture:** Fully optimized for mobile and desktop browsing.
* **Performance-First:** Minimal external dependencies for instant load times.
